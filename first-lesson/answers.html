Что такое и зачем нужен HTML?

Это язык гипертекстовой разметки, используется для написания сайтов.

Что такое и зачем нужен Git?

Git - Система контроля версий, предназначена для удобства командной работы над кодом.

Что такое SEO?

Поисковая оптимизация. Комплекс мер для поднятия сайта в поисковиках.

В чём отличие Git от GitHub?

Git - это сама программа, которая осуществляет контроль версий. Она создаёт репозитории, которые нужно где-то хранить. А GitHub - это сайт, где можно хранить репозитории, созданные в Git.

Что такое тэг? Что такое атрибут?

Тэг - это именованная метка, элемент языка разметки гипертекста. Атрибут - свойство тэга, дающее дополнительные возможности форматирования. Записывается в виде имя атрибута=значение, текстовые значения заключаются в кавычки.

Чем отличается язык разметки от языка программирования? HTML можно назвать языком программирования? Аргументируйте свою позицию.

HTML - это гипертекстовый язык разметки, не язык программирования. Он используется для приданию тексту структуры.

Зачем нужен атрибут alt у изображений?

Атрибут alt устанавливает альтернативный текст для изображений, который оттображается в случае проблем с загрузкой изображения по той или иной причине.

Зачем нужен тег label? Как объеденить label и input?

Тег label создает связь между меткой и элементом формы. Для того, чтобы связать объект и метку, нужно либо использовать идентификатор id внутри элемента формы и указать его имя в качестве атрибута for тега label, либо поместить элемент внутрь контейнера label.

Как объеденить радиокнопки в группу?

Для этого нужно указать одинаковый атрибут name для всех радиокнопок в группе.

Какие условия нужно соблюсти, чтобы отправить файл на сервер?

Для этого необходимо указать адрес программы или документа, который обрабатывает данные файлы, в атрибуте action и указать метод,который браузер использует для отправки файла, а также определить способ кодириования параметром enctype.

Что такое HTTP? Какие существуют HTTP-методы?

HTTP - hypertext transfer protocol. Протокол передачи данных, изначально предназначенный для передачи гмпертекстовых документоа. Основные HTTP-методы - POST, GET, PUT, DELETE.

Зачем используется атрибут name?

Атрибут name определяет уникальное имя элемента формы, которое, как правило, используется для отправки данных на сервер.

Что такое семантика? Зачем нужна?

Семантика - это смысл, информационное содержание кода или его отельной части. Использование семантических тегов улучшает читаемость кода поисковиками, для человека код становится интуитивнее и понятнее, также код становится более гибким, т.к. проще получить доступ к элементу.

В каких случаях используется тег article?

Используется для вставки боковых панелей, рекаламы, групп элементов nav и другого содержимого, что воспринимается отдельно от содержимого страницы.

Зачем используются ARIA-роли и ARIA-атрибуты?

Они используются, чтобы сделать сайт более доступным для людей с ограниченными способностями.

Зачем используются теги iframe, audio, video?

Iframe - рамка на основном документе, в область которой можно загружать другие документы. Audio, video - встраивает в документ проигрыватель аудио/видео соответственно, которые могут проигрывать заданные файлы.

Какие хорошие и плохие стороны формата JPEG?

Плюсы - реалистичность, высокая и управляемая степень сжатия, читается всеми браузерами, при разумной степери сжатия предоставляет хороший уровень качества при невысоком размере. Минусы - при сжатии могут вылазит различные артефакты (помехи, квадратики и т.д.), не поддерживает прозрачность.

Какие отличия PNG-8 от PNG-24?

В цветности палитры (256 против порядка 16000000) и размере.

Для чего может использоваться формат GIF?

Для оживления сайтов, в качестве анимированных элементов (баннеров, кнопок и т.д.)

Что такое векторная графика? Какие ее преимущества перед растровой?

Векторная графика состоит из объектов, описанных математически. Это позволяет масштабировать изображение без потери кач-ва. Размер файла векторной графики гораздо меньше растровой. Также, в отличие от растровой, векторная графика содержит альфа-канал, что позволяет добавлять прозрачность.

Для чего используется сервис Tinypng и подобные ему?

Для простого и удобного сжатия, оптимизации и конвертации графических файлов.

В чем преимущества Avocode по сравнению с Photoshop?

Низкий порог вхождения, оптимизация, удобство работы.
